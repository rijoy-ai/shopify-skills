# Organic traffic → conversion playbook

For `seo-traffic-conversion-audit`.

---

## 1. Vanity traffic signals

| Signal | Meaning |
|--------|---------|
| High sessions, 0 ecommerce | Vanity or broken path |
| High bounce, no PDP views | No bridge to shop |
| Long time on page, no add-to-cart | Content works; commerce missing |

---

## 2. Intent vs placement

| Intent | Placement style |
|--------|-----------------|
| How-to / tutorial | "Tools we use" → product block |
| Comparison | Table with your SKU column |
| Listicle | One slot owned by hero product |
| Evergreen guide | Sticky "Shop this look" + mid H2 |

---

## 3. Internal link rules

- First commercial link **after** value delivered (not above fold spam).
- **Descriptive anchor** (not "click here").
- One **collection** + one **hero PDP** per long post often enough.

---

## 4. What not to do

- Stuff keywords for rank (rank skill territory).
- Promise rankings from placement alone.

# 指标口径与看板字段（高复购小件）

目标：让“运营/投放/内容/客服”对同一套指标说同一种语言，避免复盘对不齐。

---

## A. 核心指标（建议周看）

### 1) 流量与转化漏斗
- **曝光**：商品/店铺/内容被看到的次数（平台定义为准）
- **点击率 CTR**：点击 / 曝光
- **加购率**：加购人数 / 访客数（或加购次数/访客，需统一）
- **收藏率**：收藏人数 / 访客数
- **支付转化率 CVR**：支付买家数 / 访客数
- **客单价 AOV**：支付金额 / 支付订单数（或/支付买家数，需统一）

### 2) 交易与质量
- **GMV**：支付金额（是否含退款、是否含运费，以平台口径为准）
- **毛利**：GMV - 商品成本 - 平台/支付/履约成本（若无法精算先用毛利率估算）
- **退款率**：退款订单数 / 支付订单数（可加“仅已完成发货口径”）
- **售后原因分布**：不适配/质量/色差/物流/期望落差/过敏等（标签化）

### 3) 复购与老客
- **老客占比**：老客支付金额 / 总支付金额（或老客订单数占比）
- **二单率**：首购用户在 X 天内产生第2单的人数 / 首购人数
- **30天复购率**：当期下单用户在 30 天内再次购买的人数 / 当期下单人数
- **复购周期**：再次购买的中位数天数（建议按品类分开）

### 4) 口碑与内容
- **评价率**：评价订单数 / 支付订单数
- **晒图/视频评价率**：晒图或视频评价订单数 / 评价订单数
- **差评率**：1-2星评价订单数 / 评价订单数
- **内容转化**：内容引流成交金额 / 内容引流访客数（平台有不同口径）

---

## B. 复购体系的分层字段（RFM 简化）

建议为每个用户打标签：
- **Recency**：最近一次购买距今天数
- **Frequency**：近90天购买次数
- **Monetary**：近90天累计消费金额（或毛利贡献）

输出字段示例：
- user_id / platform_uid
- last_purchase_date
- days_since_last_purchase
- orders_90d
- gmv_90d
- segment（新客/活跃/高价值/沉默/流失）
- preferred_category（如手机壳/贴膜/配件；或口红/底妆/护肤）

---

## C. 复盘必看的“最小数据集”（拿不到全量也能复盘）

每周至少要拿到：
- 按天：访客、支付买家、GMV、转化率、客单价、退款率
- TOP SKU：销量、GMV、毛利率估计、退款率、评价星级
- 老客：老客占比、二单率或30天复购率（取其一）
- 评价：评价率、差评率、差评原因Top3
